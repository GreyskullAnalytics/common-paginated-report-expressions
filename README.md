# Paginated Reports - Expression Reference

> **A free resource by [Greyskull Analytics](https://www.greyskullanalytics.com)**  
> *Data Solutions that make businesses better*

A reference library of common expressions for Power BI Paginated Reports (SSRS), organised by category. Designed to save you from hunting through documentation every time you need a date range, a formatting trick, or a conditional expression.
